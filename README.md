# Juris Power

Juris Power é uma landing page sobre uma advocacia, com um layout moderno e responsivo, desenvolvido em React.js e Next.js. É o meu primeiro projeto feito utilizando Styled Components, com o intuito de entender como se usa a tecnologia na prática.

## Bibliotecas utilizadas

- **styled-components**: Para estilização direto em componentes.
- **swiper**: Para criação de slides dinâmicos.
- **AOS**: Para animações.
